# Emotion Recognition Project - FER2013
Developed by Ophir Adiv

## Project Summary
This project implements a Convolutional Neural Network (CNN) to classify facial expressions into 7 emotion categories. The goal was to create a robust model capable of real-time inference.

## Final Results
- **Test Accuracy:** 61.62% (Evaluated on 7,178 unseen images).
- **Performance:** The model demonstrates strong generalization and high reliability on the FER-2013 test set.

## Implementation & UI
The project includes a web-based application built with **Gradio**. 
- **Features:** Instant prediction, large emoji visual feedback, and confidence score distribution.
- **Tools:** Python, TensorFlow/Keras, OpenCV, and Gradio.
